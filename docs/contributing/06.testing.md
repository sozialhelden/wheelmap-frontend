# Testing

For automated testing, we use [PlayWright](https://playwright.dev) (which enables quick E2E tests locally and on CI) and [BrowserStack](https://browserstack.com) (for testing manually with [BrowserStack Live](https://www.browserstack.com/live)). We thank the PlayWright and BrowserStack teams for their great products and their support! ❤️

### Getting started

Test files: **/*.e2e-spec.ts   
Tests use the CI_TEST_DEPLOYMENT_BASE_URL environment variable as the base URL

To test locally,
- [Install the PlayWright VSCode extension](https://playwright.dev/docs/getting-started-vscode),
- Run `npx playwright install --with-deps` to install PlayWright's browser binaries,

Run the tests:  
`npm run dev`  # App runs at http://localhost:3000

In another terminal, run the Playwright tests  
`CI_TEST_DEPLOYMENT_BASE_URL=http://localhost:3000 npm run test:e2e`

Or for a specific test  
`CI_TEST_DEPLOYMENT_BASE_URL=http://localhost:3000 npx playwright test src/modules/search/tests/search-input.e2e-spec.ts`

View the test report  
`npx playwright show-report`
